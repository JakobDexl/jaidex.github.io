## Running locally

You need Ruby and gem before starting, then:

```bash
# install bundler
gem install bundler

# clone the project
git clone https://github.com/JakobDexl/JakobDexl.github.io.git
cd 'foldername'

# run jekyll with dependencies
bundle exec jekyll serve
```
